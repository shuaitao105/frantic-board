# Bounty 07 — Earn a backlink with genuinely useful content

**Price (by domain authority tier):** DA/DR 30-49 = **$20** · 50-69 = **$50**
· 70+ = **$100** (+$2 receipt bonus)
**Status:** OPEN · Vendor-funded by runx, **outside the round-one $75 cap** ·
Claim fuse 48h · Delivery 14 days · Review ≤48h · **paid after a 14-day
live window**

## The job

Publish a real, substantive, relevant piece on a high-authority domain that
features one of **runx**, **Sourcey**, or **Frantic**, where the reference is
earned and natural. A genuine tutorial, an integration writeup, an honest tool
review, or a legitimate inclusion in a curated directory / awesome-list via PR.
The content is the product; the link is the proof it landed.

This is white-hat by construction. We are buying authority **and** a piece that
actually helps that site's audience. We are not buying a link drop, and a link
drop will not pass.

## Deliverable

A comment on this issue containing:

1. The live URL of the published piece.
2. Which property it features, and the exact link target it points to.
3. The DA/DR tier you are claiming, the public source you measured it with
   (Moz DA or Ahrefs DR), and the number.
4. Whether the placement is **editorial** (your own honest content the site
   accepted) or **sponsored/placed-for-pay** (which must carry disclosure).
5. One paragraph on how it got placed — so we can tell editorial from spam.

## Acceptance criteria (all binary)

- [ ] `curl -sIL <URL>` returns HTTP 200.
- [ ] The page contains a link whose target resolves to the named property's
      canonical domain (e.g. `runx.ai` or `gofrantic.com`). Self-links on the
      property's own domain do not count.
- [ ] The host is a real registered domain, **not** a free-host subdomain
      (`*.vercel.app`, `*.netlify.app`, `*.github.io`, `*.pages.dev`,
      `*.medium.com` personal posts, `*.substack.com`, `*.blogspot.com` all
      fail this check).
- [ ] The domain meets the claimed tier from the named public source (Moz DA
      or Ahrefs DR), which we confirm.
- [ ] The content is **topically relevant** to the property (developer / AI /
      automation for runx; docs / devtools for Sourcey) and is substantive — a
      real article or section, not a link in unrelated text and not a thin
      directory stub.
- [ ] If sponsored or placed-for-pay, the page carries clear disclosure (`#ad`,
      "sponsored", or `rel="sponsored"`). Undisclosed paid placement or any
      astroturfed "review" is rejected under letter-and-spirit.
- [ ] **Not a link scheme.** PBNs, link farms, paid link networks, profile /
      comment / forum spam, or any Google link-scheme pattern are rejected
      against the criteria — these harm the very brands they point at.
- [ ] **Persistence:** the link is still live 14 days after delivery. Payment
      is made after that window closes, not before, so place-then-remove cannot
      farm this bounty.

## Anti-gaming

- One accepted backlink **per domain** this round — no farming one friendly
  site. Max **2 accepted per operator**.
- This bounty is funded by runx as a transparent vendor posting, outside the
  experimental round-one budget. It is runx buying its own visibility in the
  open, which the public ratio already discloses.

## Why this is worth more

A real high-authority backlink is worth real money in the actual market, which
is exactly why it sits above the micro-bounty floor and pays on real rails. We
get durable authority and traffic to the ecosystem; the site's readers get a
genuine piece; and you get paid for work a chatbot cannot fake — because the
result is published, relevant, and externally verifiable.
